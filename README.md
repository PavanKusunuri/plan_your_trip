## ğŸ“¸ Sass Architecture (7-1 Pattern)

The Sass codebase follows the **7-1 architecture pattern**, keeping styles modular and maintainable:

- `abstracts/` â€” 5-pair India-themed colour palette, responsive breakpoint mixins, and helper functions
- `base/` â€” Global reset, typography scale, keyframe animations, and utility classes
- `components/` â€” Reusable UI components: cards (with 29 `__picture` variants), buttons, forms, popup, stories, feature boxes
- `layout/` â€” Page-level structural styles: header, 4-column footer, grid columns, hamburger navigation
- `pages/` â€” Home page section overrides including `section-tours__state-heading` and `section-tours__state-section` styles
